## Getting Started


Clone the repository

- With SSH

```bash
git clone git@github.com:username/ReactCalcutator.git
```

- With https
```bash
git clone https://github.com/username/ReactCalcutator.git
```

Install the dependencies:

```sh
$ npm install
// or
$ yarn
```

Run in dev mode:

```sh
$ npm run dev
// or
$ yarn dev
```

### Built With

- JavaScript
- [Vite](https://vitejs.dev)
- [React](https://reactjs.org/)
